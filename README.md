# 🔥 Fire & Smoke Detection

A Python based Deep Learning project for detecting **fire** and **smoke** using trained model weights.

This project aims to provide an early warning system by identifying fire and smoke using computer vision and machine learning techniques.

---

## 📌 Features

- 🔥 Fire detection
- 💨 Smoke detection
- 🧠 Pre trained model weights included
- 🚀 Ready for experimentation and improvements

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/kaushal033/fire-smoke-detection.git
cd fire-smoke-detection
```

### 2. Create a virtual environment (Recommended)

### Conda

```bash
conda create -n fire-smoke python=3.11
conda activate fire-smoke
```

### OR venv

**Windows**

```bash
python -m venv venv
venv\Scripts\activate
```

**Linux / Mac**

```bash
python -m venv venv
source venv/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Usage

Run the detection script:

```bash
python main.py
```

The program loads the trained weights and performs fire and smoke detection based on the configured input.
